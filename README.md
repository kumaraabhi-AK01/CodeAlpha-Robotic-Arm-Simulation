# CodeAlpha Robotics & Automation Internship

## Task 2: Robotic Arm Simulation

### Project Overview
This project demonstrates the design and motion simulation of a 7-Degree of Freedom (7-DOF) industrial robotic arm using Autodesk Fusion. The robotic arm was designed with revolute joints and configured to perform a basic pick-and-place operation.

### Objectives
- Design a 7-DOF industrial robotic arm.
- Define the Degrees of Freedom (DOF).
- Create revolute joints for realistic movement.
- Perform a basic pick-and-place motion simulation.

### Software Used
- Autodesk Fusion (Education License)

### Robot Specifications
- Degrees of Freedom (DOF): 7
- Joint Type: Revolute
- Motion: Pick-and-Place Simulation
